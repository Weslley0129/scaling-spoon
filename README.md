# Entrega Final - Qualidade e Teste de Software

Repositorio com os artefatos da avaliacao final.

## Estrutura principal

- `web-automation/`: automacao Web com Cypress + Cucumber
- `api-automation/`: automacao de API com Postman + Newman
- `final-avaliacao/docs/`: plano de testes, evidencias e PDF para upload

## Web Automation

Pasta: `web-automation`

- Base URL alvo: `https://automationexercise.com`
- Automacao de fluxo de pedido em Gherkin (`.feature`)
- Step definitions com Cypress + Cucumber
- Instrucoes de execucao no `web-automation/README.md`

## API Automation

Pasta: `api-automation`

- APIs alvo: `https://dummyjson.com`
- Colecao Postman com cenarios de:
  - produtos (add/update/delete)
  - usuarios (add/update/delete)
  - autenticacao (captura token + validacao do usuario)
- Instrucoes de execucao via Newman no `api-automation/README.md`
