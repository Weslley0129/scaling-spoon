# Entrega Final - Qualidade e Teste de Software

Este pacote contem os artefatos solicitados para avaliacao final:

- `docs/`:
  - `plano-de-testes-web.md` (7 suites, 3 casos por suite)
  - `evidencias-execucao-manual.md` (modelo de registro de execucao)
- `web-automation/`:
  - Cypress + Cucumber para fluxo de realizacao de pedido
  - README de execucao
- `api-automation/`:
  - Colecao Postman + Environment para DummyJSON
  - Execucao automatizada via Newman
  - README de execucao

## Passos recomendados para finalizar a entrega
1. Subir `web-automation` em um repositorio GitHub.
2. Subir `api-automation` em outro repositorio GitHub (ou no mesmo, em pastas separadas).
3. Criar conta trial no Testomat e cadastrar os 21 casos de teste (7 suites x 3 casos).
4. Executar manualmente ao menos 1 ciclo e anexar evidencias.
5. Executar automacoes (Cypress e Newman) e gravar video curto mostrando:
   - casos de teste,
   - codigo,
   - execucao.
6. Compartilhar os links de GitHub + Testomat com o tutor.
